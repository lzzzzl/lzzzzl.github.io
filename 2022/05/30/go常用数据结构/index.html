Go 常用数据结构

2022-05-30

引言

在 Go 中，与 Java 不同的是，没有浩繁且功能多种多样的集合类。在日常开发中，主要用的是 Array、Slice、Map 以及 List。下面，让我们来一起分析这几个数据结构。

数组

Go 语言中的数组与其他语言的数组十分类似，数组的底层是计算机为其分配的一块连续内存，在连续内存中保存了数组的元素。因此，数组查询快，只需利用数组中元素的索引即可快速访问元素对应的存储地址。增删慢，因为插入和删除涉及到数组内存的重新分配。

数组的初始化

初始化后的数组:

同时，数组不需要被初始化，整型数组的默认值是零，字符串数组的默认值为空，布尔数组的默认值为false。

Go 的数组是值。一个数组变量代表一个数组，而不是指向数组第一个元素的指针（C中是这样）。这意味着，当我们分配或传递一个数组变量时，会将该数组的内容进行复制。因此，当数组被赋予给一个新变量时，在新变量中所做的更改不会影响到旧变量。

在数组中，如果数组元素是可比的，那么数组类型也是可比的。因此，我们可以直接比较两个数组。

切片的创建

1. 使用 make 创建切片

使用内置函数 make 创建一个切片。make 函数接受类型、长度和可选的容量。当调用时，make 分配一个数组并返回引用这个数组的切片。

3. 切片和数组创建的不同

切片的类型规范是 []T，其中 T 是切片元素的类型。与数组不同的是，切片类型没有指定长度。

零值切片

零值的切片为 nil。 在声明时不做初始化，就会创建一个 nil 切片。

零值切片的 len 和 cap 都返回 nil
